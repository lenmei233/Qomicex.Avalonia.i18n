# Qomicex.Avalonia.i18n
Welcome to the localization hub for Qomicex Minecraft Launcher! This repository is dedicated to managing the JSON language files used in our launcher.
​We warmly invite community contributors to help translate and localized Qomicex for Minecraft players worldwide. Whether you are fixing a typo or adding a brand new language, every contribution is highly valued.

​欢迎来到 Qomicex Minecraft Launcher 的本地化协作中心！这里专门用于托管启动器所需的 JSON 多语言资源文件。
​我们诚挚地邀请社区贡献者参与翻译，帮助我们将 Qomicex 带给全球更多的 Minecraft 玩家。无论你是修复一个拼写错误，还是新增一个完整的语言支持，你的每一份贡献都至关重要。
